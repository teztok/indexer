# Tags

Tags are metadata labels associated with tokens, extracted from the token metadata. They help categorize and discover tokens based on their characteristics or themes.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| fa2_address | string | The contract address of the token (always starts with "KT"). |
| token_id | string | The ID of the token. |
| tag | string | The name of the tag. |

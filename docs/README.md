# TezTok Indexer Documentation

This directory contains the documentation for the TezTok indexer in Markdown format.

WARNING: this documentation is not complete and can rely on things that don't exist anymore since teztok services are down. It should still provide enough information to help you start to use this indexer.

## Documentation Structure

- [Introduction](index.md) - Overview of the TezTok indexer and supported marketplaces
- [Installation](installation.md) - Setup instructions for running the indexer
- [Architecture](architecture.md) - Technical architecture and service design
- [Principles](principles.md) - Core design principles (idempotency and statelessness)
- [Events](events.md) - TezTok event system and event types
- [Tokens](tokens.md) - Token data model and properties
- [Holdings](holdings.md) - Token ownership tracking
- [Listings](listings.md) - Active marketplace listings
- [Offers](offers.md) - Bid and offer tracking
- [Tags](tags.md) - Token categorization and metadata tags
- [Royalty Receivers](royaltyreceivers.md) - Royalty distribution tracking
- [Plugins](plugins.md) - Plugin system for extending the indexer
- [Maintenance](maintenance.md) - Operating and maintaining the indexer
- [Examples](examples.md) - Common GraphQL query examples
- [FAQ](faq.md) - Frequently asked questions

## Getting Started

Start with the [Introduction](index.md) to learn about the TezTok indexer, then follow the [Installation](installation.md) guide to set up your own instance.
